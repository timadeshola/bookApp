# bookApp
Test-Driven backend application to display list of books and their average rating

Stack: Spring Boot

Modules:

User Management
- Create User
- Update Existing User
- Delete User
- Toggle User Status
- View All Users
- View User by ID
- View User by Username

Role Management
- Create Role
- Update Role
- Delete Role
- Toggle Role Status
- View All Roles
- View Role By ID
- View Role By Role Name

Book Management
- Create Book
- Update Book
- Delete Book
- Toggle Book Status
- View All Books
- View Book by Author
- View Book by Title
- View Book by ISBN Number

Book Rating
- Rate A Book
- View Rating Average
